현실 - 그림 그릴 때 필요한 요소
	화가 - 그림의 주체 : 인간
	팔레트 - 색상 처리 : 도구
	붓 - 그리는 "행위"에 사용 : 도구
	캔버스 - 그려질 대상 : 대상

전산 - 자바 프로그래밍에서의 GUI 요소
	그림의 주체 : component 자체.
	그리는 행위 : paint method.
	색상 처리 : Graphics라는 이름으로 paint method의 매개변수에 존재.
		자바에서의 팔레트는 색, 도우미, 모양 등 다양한 것을 가지고 있는 객체이다.
	그려질 대상 : component 자기 자신. == 메이크업과 비슷.

gui 프로그램 설계시 순서(상황에 따라 다를 수 있음).
	메인이 되는 컴포넌트를 상황에 맞도록 extends 한다.
	필요한 모든 요소들을 멤버변수로 정의한다. (import도 한다)
	객체가 만들어질 대 요소들이 같이 만들어질 수 있도록, 생성자 함수에 멤버변수들을 new 한다.
	각 요소 배치와 설정을 한다.
	이벤트 발생 시 동작하는 것이 있다면, 
		eventListener를 메인 컴포넌트(자기자신)에게 implements 하거나,
		다른 Listener 전용 클래스를 만등러 implements한다.
	리스너 클래스를 정의하고 연결.
	main()에 메인이 되는 class를 new 한다.
