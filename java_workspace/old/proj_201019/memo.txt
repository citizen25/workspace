전역변수는 프로그램이 끝날때까지 살아있는다.

환경변수란?
	os에 등록되는 전역변수
	그 중에서도 path라는 시스템 환경변수의 역할
		전역적 경로 참조
		따라서 path에 등록된 경로에 해당하는 파일은 어디서든 참조가 가능하므로,
		full 경로를 명시하지 않아도 된다.


시스템의 환경경변수인 path에 경로를 등록해놓으면,
해당 위치가 전역 변수로 등록되어 있으므로,
어느 위치에서나 참조가 가능하다.

[ 자바 cmd에서 실행하기 ]
	기계어로 컴파일 : javac.exe 실행. -> 하기 전에 환경변수로 javac.exe의 경로(%java_home%\bin)를 넣어준다.
		javac Test.java 를 입력하면, .class 확장자의 파일이 생긴다. (기계가 이해할 수 있는 파일)
	파일 실행 : java.exe로 실행 가능 
		java Test 입력 (Test.class 라 입력하지 않는다.)
